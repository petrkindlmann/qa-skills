AI-Augmented QA

Four skills for the intersection of AI and quality assurance — not available in any other skills collection.

ai-test-generation
Staged pipeline: requirements → risks → coverage matrix → scenarios → oracles → test code → human review. With guardrails against common AI failures.
ai-bug-triage
Hybrid fingerprint + LLM pipeline. Deterministic deduplication first, then LLM for severity classification and ticket generation.
test-reliability
Multi-attribute selector healing, environment-aware diagnosis, flake classification, confidence-scored auto-repair with observable workflows.
ai-qa-review
Test quality assessment across five smell dimensions — readability, reliability, diagnostics, design, and coverage. Each linked to a review action.
